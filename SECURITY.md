# Security Policy / 安全政策

## Supported Versions / 支持版本

| Version | Supported |
|---------|-----------|
| 0.1.x   | ✅        |

## Reporting a Vulnerability / 报告漏洞

**请不要通过公开 Issue 报告安全漏洞。**

如发现安全问题，请通过以下方式私下联系：

- GitHub：发送 [Private Security Advisory](https://github.com/xuanyustudio/papersketch-js/security/advisories/new)
- 或直接通过仓库联系作者

收到报告后，我们会在 **7 个工作日内**确认并评估，修复后在 Changelog 中注明（不披露报告者信息，除非获得同意）。

---

**Do not report security vulnerabilities via public Issues.**

Please use [GitHub Private Security Advisories](https://github.com/xuanyustudio/papersketch-js/security/advisories/new) or contact the maintainer directly.

We aim to acknowledge reports within **7 business days**.
